Project Name: COVID-19 and Unemployment Analysis

Description:
This project aims to analyze the relationship between COVID-19 cases and unemployment rates using two datasets. 
The project involves cleaning the datasets and performing exploratory data analysis to examine any potential correlations between the variables. 
The analysis is conducted using Jupyter Notebook and Python.

Files:
1. CovidCases.csv: This dataset contains information about daily COVID-19 cases in different regions.
2. UnemploymentRate.csv: This dataset contains information about monthly unemployment rates in the same regions.
3. analysis.ipynb: Jupyter Notebook file that performs the correlation analysis between COVID-19 cases and unemployment rates.

Dependencies:
- Python 3.1
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
